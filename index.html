About Nebula Nexus

Nebula Nexus is a comprehensive database for EVE Online exploration and combat sites. This tool helps capsuleers quickly find and filter sites based on safety, type, region, and difficulty.

Data Sources

  • EVE University Wiki - Gas Cloud Harvesting
  • EVE University Wiki - Relic and Data Sites
  • EVE University Wiki - Combat Sites & Anomalies

Features

  • Comprehensive exploration sites database
  • Complete combat anomalies listing
  • Advanced filtering system
  • Safety indicators for all sites
  • Quick filter chips for common searches

User Guide

Safety Indicators

  • Safe: No NPCs present, can be done safely with exploration frigates
  • Unsafe: Contains hostile NPCs that will attack on sight
  • Warning: NPCs spawn after timer or contain special mechanics

Exploration Sites

  • Gas Sites: Harvest gas clouds for profit. Wormhole gas (Fullerite) is most valuable
  • Relic Sites: Hack containers for salvage materials and blueprint copies
  • Data Sites: Hack containers for datacores, decryptors, and blueprints

Combat Anomalies

  • Hideaway (Difficulty 1): Easiest sites, low bounties
  • Burrow/Den (Difficulty 2-3): Easy-Medium difficulty
  • Yard/Rally Point (Difficulty 4-6): Medium-Hard difficulty
  • Port/Hub (Difficulty 7-8): Hard difficulty
  • Haven/Sanctum (Difficulty 9-10): Hardest sites, best rewards

Using Filters

  • Use the search box to find specific site names
  • Select dropdowns to filter by safety, type, region, or faction
  • Click quick filter chips for instant common filters
  • Click table headers to sort by that column
